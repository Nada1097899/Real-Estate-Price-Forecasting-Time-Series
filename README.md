#  Real Estate Price Forecasting & Regression Analysis

An advanced data science project focused on predicting median sale prices in the housing market using time-series data and sophisticated regression algorithms.

##  Project Overview
Predicting property prices requires more than just simple averages. This project implements a robust pipeline to clean, transform, and model real estate data, accounting for temporal trends and market fluctuations.

##  Tech Stack & Advanced Techniques
* **Machine Learning:** Gradient Boosting, XGBoost, Random Forest, Linear Regression.
* **Outlier Detection:** Isolation Forest (to remove anomalous price points).
* **Missing Data Imputation:** KNNImputer (for intelligent value replacement).
* **Feature Transformation:** Log Transformation to stabilize variance and normalize target distribution.
* **Libraries:** Scikit-learn, XGBoost, Pandas, Seaborn, Matplotlib.

##  Key Technical Implementation
- **Data Refining:** Successfully handled missing dates and prices through logical imputation.
- **Dimensionality Management:** Explored PCA for feature reduction while maintaining predictive power.
- **Model Tuning:** Evaluated performance using RMSE (Root Mean Square Error) and R² Score.

##  Performance Results
- **Best Model:** Gradient Boosting Regressor.
- **R² Score:** 0.9569 (Explaining ~96% of price variance).
- **Visuals:** Residual plots and Actual vs. Predicted price comparisons.
